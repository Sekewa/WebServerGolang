Welcome to WebServerGolang !

It's a simple (really simple) Web Server Coded in Golang.

You can either make it with the makefile and launch it in cmd or use the dockerfile to containerize it.

If you see this page its because ou didn't put the static dir with a index.html, here is an example :

 .
├──📂bin
├──📂cmd
│   └──📁web
├──📂example
│   ├──📁css
│   └── index.html
├──📂internal
│   ├──📁 handlers
│   └──📁 utils
├──📁static (👈you need to create this one)
├──.gitignore
├──go.mod
├──Makefile
└──README.md

If you want to put your own site you need to put it in the static folder.
This way the web server will know it need to use it.
